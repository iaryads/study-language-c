# 💻 Estudos de Lógica de Programação em C
Este repositório registra minha evolução técnica como estudante de **Análise e Desenvolvimento de Sistemas (ADS)**. 
Aqui documento desafios de lógica, sintaxe e algoritmos desenvolvidos durante minha jornada.

## 🛠️ Tecnologias e Conceitos
- **Linguagem:** C
- **Conceitos explorados:** Estruturas de repetição (for/while), Condicionais (if/else), Aritmética Modular.

## 🚀 Projetos em Destaque
### 1. Simulador de Cronômetro Alternado (Tick-Tack)
Um algoritmo interativo que utiliza paridade numérica para alternar mensagens no terminal.

- **O que aprendi:** Uso prático do operador `%` (módulo) e controle de fluxo com `for`.
- **Desafio superado:** Ajustar a lógica para que o contador fosse baseado no incremento do laço e não em um valor estático.
